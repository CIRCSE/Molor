# MOLOR_Lemma-Bank

The MOLOR Lemma Bank currently consists of approximately 6000 Old Irish lemmas, covering nouns, numerals and verbs.
This Lemma Bank serves as the backbone of the incipient knowledge graph for Old Irish, where entries in lexical resources and tokens in corpora point to the same lemma. For more information, see [molor.eu](https://molor.eu).

Data is avalable as a graph database (RDF triples - Turtle format). For quering the data, use the SPARQL endpoint at [molor.eu/sparql](https://molor.eu/sparql).
